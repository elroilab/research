# HSIFoodInspection: A Comprehensive Hyperspectral Dataset for Industrial Food Anomaly Detection.
> Chi Zhang, Jungkwon Kim, Jihun Kim, Jeonghyeon Park, Kwangsun Yoo, Seok-Joo Byun

## About
Official repository for dataset and template code sharing for *HSIFoodInspection*, accepted at the CVPR MetaFood Workshop 2026.

> [!NOTE]
> This repository is actively being updated. Dataset link, manual, and template code will be released soon.
